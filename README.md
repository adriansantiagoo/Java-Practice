# Java-Practice

This repository contains my daily practice exercises and solutions for mastering Java fundamentals, Object-Oriented Programming (OOP), and data structures. It serves as a personal log of my coding progress and a portfolio of my problem-solving skills.

## 🛠️ Technologies Used
* **Language:** Java 21
* **IDE:** IntelliJ IDEA
* **Build System:** IntelliJ Native

## 📁 Project Structure
The project is organized into packages inside the `src/` directory, with each package focusing on a specific topic:

*   `src/Collections/` — Dynamic arrays, basic CRUD operations, and sorting lists.
*   `src/DataStructures/` — Custom implementations of Stacks, Queues, and Linked Lists.
*   `src/OOP_concepts/` — Inheritance, Polymorphism, Abstraction, and Encapsulation exercises.
*   `src/ProgrammingLogic/` — Exercises to practice my logic

## 🚀 How to Run the Code
To run these exercises locally on your machine:

1. Clone the repository:
   ```bash
   git clone https://github.com
   ```
2. Open **IntelliJ IDEA**.
3. Select **File > Open** and choose the cloned repository folder.
4. Navigate to the specific package and file you want to test (e.g., `src/arraylists/Main.java`).
5. Right-click inside the file and select **Run 'Main.main()'**.
